ЩО ВСЕРЕДИНІ :

Крок 1. Скраб для губ м'яко відшулушує суху шкіру губ
Нанесіть скраб, потім просто помасажуйте пальцем, щоб відлущити суху або огрубілу шкіру. Змийте скраб, що залишився, водою. Використовуйте як частину вечірнього догляду або перед нанесенням матової помади. Використовуйте бальзам для губ Girlwood відразу після скрабу.

Крок 2. Бальзам для губ зволожує та надає природний блиск
Наносьте кожного разу після скрабу або в будь-який час дня, коли ваші губи потребують додаткового зволоження.

Крок 3. Маска загоює тріщини за допомогою оливкового масла
Використовуйте маску як частину вашого ранкового догляду або за 20 хвилин до нанесення матової помади. Немає необхідності змивати маску.

Крок 4. Олійка для губ - живить та зволожує.
Використовуйте олію для губ протягом дня, коли губи потребують зволоження..
